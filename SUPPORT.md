# Support

## Getting Help

If you need help with Bolt Spring Boot Starter, here are your options:

### Documentation

- [README](README.md) — Quick start guide, configuration reference, and annotation documentation
- [Slack Bolt for Java docs](https://docs.slack.dev/tools/java-sdk-bolt/) — Upstream Bolt SDK documentation

### Questions

- **GitHub Discussions / Issues**: Open a [question issue](https://github.com/jwcarman/bolt-spring-boot/issues/new?template=question.yml) on GitHub

### Bug Reports

- Open a [bug report](https://github.com/jwcarman/bolt-spring-boot/issues/new?template=bug_report.yml) with reproduction steps

### Feature Requests

- Open a [feature request](https://github.com/jwcarman/bolt-spring-boot/issues/new?template=feature_request.yml) describing your use case

### Security Issues

- See [SECURITY.md](SECURITY.md) for responsible disclosure instructions
